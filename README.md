###### 📘 EduNexa – Learning Management System (LMS)

###### 

###### EduNexa is a full-featured Learning Management System (LMS) designed to deliver industry-aligned courses, online exams, and digital certifications with separate Admin and Student portals.

###### 

###### 🚀 Features Overview

###### 👨‍🎓 Student Features

###### 

###### Student registration \& login

###### 

###### Browse and enroll in courses

###### 

###### Learn via video \& PDF content

###### 

###### Take quizzes and final exams

###### 

###### Exam pass/fail logic

###### 

###### Download digital certificates

###### 

###### Student dashboard with progress tracking

###### 

###### Resume building \& interview preparation

###### 

###### 🔐 Admin Features

###### 

###### Secure admin login

###### 

###### Create, edit \& delete courses

###### 

###### Upload:

###### 

###### 🎥 Video lessons

###### 

###### 📄 PDF materials

###### 

###### 🖼 Course thumbnails

###### 

###### Set course price (Free / Paid)

###### 

###### Create quizzes \& exams

###### 

###### Control:

###### 

###### Pass percentage

###### 

###### Exam attempts

###### 

###### Role management (Admin / Instructor / Student)

###### 

###### Certificate generation control

######

###### Frontend
